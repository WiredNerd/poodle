# Poodle

Poodle is an tool for performing Mutation Testing on Python projects.

## Features

* Multi-Threaded Mutation Testing
* Configurable
* Pluggable

## Quick Start

Installation:

```
pip install poodle-test --upgrade
```

Running:
```
poodle
```
